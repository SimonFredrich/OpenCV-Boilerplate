# OpenCV Boilerplate

An easy to use OpenCV template. It is written for c++ and build with cmake.

- [Installation](#installation)
- [License](#license)
- [Usage](#usage)

## Installation

Update CMakeLists.txt and .vscode files to your needs and proceed with:

```bash
mkdir build
cmake ..
cd ..
```
